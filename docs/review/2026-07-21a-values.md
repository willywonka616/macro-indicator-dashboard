## Reading this table

- **`tag: "live"`** means the value traces to a real HTTP call to FRED,
  Treasury, or IMF/DBnomics inside `scripts/fetch.py` on the run that
  produced the current `public/data.json`, AND that the freshness guard
  confirmed the source's latest observation is within its cadence
  threshold.
- **`tag: "manual_price"`** (new this round) means one input (the gold
  price) is hand-entered from `data/manual.json`, but every other
  component (the ounce count, FX-excl-gold, GDP) is still live — a
  hybrid, not a fully manual output.
- **`tag: "manual"`** means the value is hand-carried from
  `data/manual.json` in full — either because no live source exists for
  that row, or because a live source exists but failed the freshness
  guard and no partial-input fallback is possible for that metric.
- **`note`** is a short row-level caption, distinct from a panel's own
  `note` (rendered once, below all rows).
- **`asOf`** is the native period of the underlying data (a FRED quarter, a
  Treasury month, etc.), not the date this pipeline last ran.

(gold price still frozen since 2025-06, COFER still frozen since
2025-Q1). This array is empty on a fully-healthy run (confirmed locally
against the mock test's fully-live scenario); its presence here is the
correct, expected signal that both known-frozen sources are still frozen,
not a new problem.

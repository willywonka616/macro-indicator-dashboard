## Reading this table

- Both rows draw from the identical `gold_value_monthly` computation
  (live oz × live price) — they differ only in what they're divided by
  (or not divided by at all). Neither is "more correct" than the other;
  they answer different questions ("gold's value today" vs. "reserves
  including gold, on the same officially-lagged basis Dalio's own figure
  uses").
- `gold_value_current` has no chart (`history: []`) — a deliberate scope
  choice this round: `Chart.jsx`'s y-axis always formats values as a
  percentage, so a dollar-denominated series would need a frontend change
  beyond what was asked for. The headline number and its `ƒx` equation
  button are shipped; a chart is not.

  breakdown of every other row — unchanged this round.

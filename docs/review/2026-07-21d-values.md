## Reading this table

- **`tag: "projection"`** (new this round) means the value is CBO's own
  published baseline — a current-law benchmark, not this pipeline's
  forecast and not a current measurement. Distinguished visually with a
  dashed border (`Tag.jsx`) and, on its chart, a dashed line + shaded
  forward region + "PROJECTED" label (`Chart.jsx`).

  current measurement; only the chart's forward extension is CBO's
